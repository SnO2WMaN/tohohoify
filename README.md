## Development

Pass the environment variable `CHROME_EXECUTABLE_PATH`

```dotenv
# linux (example)

CHROME_EXECUTABLE_PATH="/usr/bin/google-chrome-stable"
```

```bash
yarn

vercel dev # npm i -g vercel
```
