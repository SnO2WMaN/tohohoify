## Usage

```
https://tohohoify.vercel.app/api?text=<text>&icon=<url>[&font=<font>][&fontSize=<fontSize>]
```

- `<font>`
  - Font in [Google Fonts](https://fonts.google.com/).
- `<fontSize>`
  - Font size can be set like `font-size: <fontSize>vmin;`. (default: 4)
